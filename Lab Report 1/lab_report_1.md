# Lab Report 1

## Command: "cd"

### No arguments
```
[user@sahara ~]$ cd
[user@sahara ~]$
```

### Using a directory as argument
```
[user@sahara ~]$ cd lecture1
[user@sahara ~/lecture1]$
```

### Using a file as argument
```
[user@sahara ~/lecture1]$ cd Hello.java
bash: cd: Hello.java: Not a directory
[user@sahara ~/lecture1]$
```
